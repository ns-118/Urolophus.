This scripts are used from e-dejong/OceanOmics-reference genomes repository, these scripst have been adapted to suit the needs of the genome assembly of Urolophus paucimaculatus. 

The order of the scripts follows a the workflow from edejong/OceanOmics-reference genomes.

1. newdir.sh

2. config.ini

3. hifiaf.sh

4. meryl.sh

5. genomescope.sh

6. hifiasm.sh

7. gfastats.sh

8. busco.sh

9. merqury

10. arima_mapping (previously was omnic.sh and then used a wrapper, adapted from VGP)

11. fcsgx-farm.sh

12. tiara.sh

13. bbmap.sh

14. cat.sh

15. gfastats2.sh

17. busco2.sh

18. merqury2.sh
